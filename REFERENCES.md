## Quick Commands

```bash
# Development
npm run dev              # Start dev server
npm run build            # Build for production
npm start                # Run production server

# Code Quality
npm run lint             # Check & fix linting
npm run type-check       # TypeScript check
npm run format           # Format code
npm run analyze          # Full analysis

# Database
npm run prisma:studio    # Open Prisma Studio
npm run prisma:migrate   # Run migrations
npm run prisma:generate  # Generate Prisma Client

# Testing
npm run test             # Run tests
npm run test:watch       # Watch mode
npm run test:coverage    # Coverage report
```

## Getting Help

### I'm stuck!

1. **Check [TROUBLESHOOTING.md](./TROUBLESHOOTING.md)** - Common issues
2. **Search [GitHub Issues](https://github.com/aeldarian1/CozyReads/issues)** - Similar problems
3. **Read relevant docs** - Architecture, API, Development guides
4. **Ask in [GitHub Discussions](https://github.com/aeldarian1/CozyReads/discussions)** - Ask community
5. **Create [GitHub Issue](https://github.com/aeldarian1/CozyReads/issues/new)** - Report bug

### Want to contribute?

1. **Read [CONTRIBUTING.md](./CONTRIBUTING.md)** - Guidelines
2. **Check [GitHub Issues](https://github.com/aeldarian1/CozyReads/issues?q=label%3A%22good-first-issue%22)** - Find good first issues
3. **Fork repository** - Start working
4. **Submit PR** - Share your work
