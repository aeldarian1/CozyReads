# Changelog

All notable changes to CozyReads will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [Unreleased]

### Added
- Testing infrastructure with Jest and React Testing Library
- Comprehensive API documentation
- Development setup guides
- Troubleshooting guide
- GitHub Actions CI/CD pipeline
- Dependabot configuration for automatic updates
- Docker support for local development
- Code of Conduct
- Security Policy

### Changed
- Enhanced README with better structure and documentation links
- Improved package.json with additional scripts
- Updated environment configuration

### Fixed
- Initial setup and onboarding experience

## [1.0.0] - 2024-01-15

### Added
- Initial release
- Core book management features
- Reading progress tracking
- Collections system
- Advanced search and filtering
- Statistics dashboard
- Import/Export functionality
- Beautiful responsive UI
- Dark/Light mode support
- Keyboard shortcuts
- Undo/Redo system
- Accessibility support (WCAG 2.1 AA)
- Type-safe codebase with TypeScript
- Clerk authentication integration

### Features Included
- Virtual 3D bookshelf
- Multi-source book enrichment
- Goodreads import
- Custom collections
- Reading goals
- Quote management
- Mobile optimization
- Rate limiting
- Input validation
